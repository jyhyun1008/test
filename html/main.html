Peachtart Project

About

Peachtart is a project that supports budget-friendly virtual creators in various ways. It operates an SNS site for communication within the Fediverse and develops various web apps and software to assist with activities.

피치타르트는 저예산 버츄얼 크리에이터를 다방면으로 지원하는 프로젝트입니다. 연합우주 내에서 함께 소통할 수 있는 SNS 사이트를 운영하며, 활동을 도울 수 있는 여러 웹앱과 소프트웨어를 만듭니다.

ピーチタルトは、低予算のバーチャルクリエイターを多方面で支援するプロジェクトです。Fediverse内でコミュニケーションを取れるSNSサイトを運営し、活動を支援するための様々なウェブアプリやソフトウェアを開発しています。

  • We provide solutions for promoting your content within the Fediverse. We run a Misskey server, enabling you to promote yourself from anywhere in the Fediverse.

    Fediverse内で自分のコンテンツを宣伝できるソリューションを提供しています。Misskeyサーバーを運営し、Fediverseのどこからでも自分を宣伝できるようにしています。

  • We develop features that integrate with software transmitting the VMC protocol. Currently, we are working on a project that calculates a single VMC signal from two VMC signals captured from different angles.

    VMC 프로토콜을 송신하는 소프트웨어와 맞물린 기능들을 개발합니다. 현재는 두 개의 서로 다른 각도에서 캡쳐한 VMC 신호로부터 단일한 VMC 신호를 계산하는 프로젝트를 제작하고 있습니다.

    VMCプロトコルを送信するソフトウェアと連携する機能を開発しています。現在、異なる角度からキャプチャされた2つのVMC信号から単一のVMC信号を計算するプロジェクトを制作しています。
